# BISD Framework

Bilingual STEM-Digital Integration Framework for K-5 Two-Way Immersion Programs.

## Overview

The BISD Framework is a bilingual STEM instructional framework designed to support K-5 Two-Way Immersion (TWI) programs through integrated scientific inquiry, collaborative digital learning, experiential science, and bilingual academic language development.

The framework was developed through practitioner-researcher action research conducted in a rural North Carolina bilingual education setting.

## Core Components

### IBICI
Bilingual Scientific Inquiry Cycle aligned with NGSS Science and Engineering Practices.

### MACE
Experiential scientific learning protocols designed for low-resource environments.

### TIDE
Structured collaborative technology integration for bilingual STEM classrooms.

### BISD Guide
Transferable K-5 curriculum guide for bilingual STEM instruction.

### FRAME
National dissemination and replication model for Two-Way Immersion programs.
