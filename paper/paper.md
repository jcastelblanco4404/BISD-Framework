# BISD Framework: Bilingual STEM-Digital Integration for K-5 Two-Way Immersion Programs

## Summary

The BISD Framework is an open educational framework designed for K-5 Two-Way Immersion (TWI) programs. It integrates bilingual scientific inquiry, experiential STEM learning, and collaborative digital pedagogy aligned with NGSS standards.

## Statement of Need

K-5 bilingual STEM programs often lack structured Spanish-first instructional frameworks aligned with NGSS and designed for Two-Way Immersion environments. Existing materials are typically translations of English curricula rather than native bilingual instructional systems.

This creates a gap in equitable STEM access for English Language Learners in early education.

## Framework Overview

The BISD Framework consists of five components:

- IBICI: bilingual scientific inquiry cycle
- MACE: experiential learning without laboratory dependency
- TIDE: collaborative digital learning structures
- BISD Guide: teacher implementation curriculum
- FRAME: dissemination model for scalability

## Implementation

The framework was developed and tested through practitioner action research in a K-5 bilingual classroom setting.
